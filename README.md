# AI/ML Engineer Course

**Target:** Industry-ready ML Engineer
**Duration:** ~9 months | 2 hrs/day
**Background:** Intermediate fullstack engineer, comfortable Python, minimal math

## How This Course Works

Math is never a standalone block — it's learned *when you need it*, attached to something you just built. By week 4 you've trained and deployed a model. Math clicks because you've already felt the problem it solves.

**Each week has:**
- `study-guide.md` — concise concepts + deep dive pointers
- `tutorial.ipynb` — hands-on implementation (run this, break this, learn this)

## Ground Rules

1. **Code everything.** No passive watching — every concept gets implemented.
2. **Ship early.** Deploy something by week 15. Your fullstack background makes this your edge.
3. **One Kaggle competition per phase.** Real data pressure, real feedback.
4. **Write one blog post per phase.** Compounds into inbound interest over time.
