# Week 8 — Phase 1 Capstone: End-to-End ML Pipeline

**Topic:** Put everything together. Build a complete, reproducible ML pipeline from raw data to predictions.

## The Project

Choose one of these datasets (or find your own on Kaggle):
- [Titanic](https://www.kaggle.com/c/titanic) — binary classification, great for beginners
- [House Prices](https://www.kaggle.com/c/house-prices-advanced-regression-techniques) — regression, rich feature engineering opportunities
- [Credit Card Fraud](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud) — imbalanced classification, real-world challenge

## Deliverables

1. **EDA notebook** — explore the data, visualize distributions, identify issues
2. **Feature engineering notebook** — all preprocessing decisions with justifications
3. **Modeling notebook** — at least 3 models compared, with proper cross-validation
4. **Submission** — score on Kaggle leaderboard
5. **Writeup** (300–500 words) — what you tried, what worked, what didn't, what you'd do next
