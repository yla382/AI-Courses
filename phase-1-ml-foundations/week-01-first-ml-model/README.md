# Week 1 — Your First ML Model + Linear Algebra I

**Time:** ~14 hrs total (2 hrs/day × 7 days)
**Goal:** Train, evaluate, and understand your first ML model. Learn the linear algebra that makes it work.

## Schedule

| Day | Focus                                                   | Resource                |
| --- | ------------------------------------------------------- | ----------------------- |
| Mon | What is ML? Scikit-learn setup                          | `study-guide.md` §1     |
| Tue | Train your first classifier (Iris dataset)              | `tutorial.ipynb` Part 1 |
| Wed | Linear Algebra I — vectors & matrices                   | `study-guide.md` §2     |
| Thu | How ML models use matrix math under the hood            | `tutorial.ipynb` Part 2 |
| Fri | Predictions, errors, and what "learning" actually means | `study-guide.md` §3     |
| Sat | Build a second model on a new dataset                   | `tutorial.ipynb` Part 3 |
| Sun | Review + write 3 things you'd explain to a colleague    | reflection              |

## Deliverable

By end of week: a trained classifier on a real dataset, with a short writeup of what the model is doing mathematically.

## What's Next

Week 2 dives into data exploration — you'll spend more time on the data *before* training, and learn the statistics behind why data quality matters so much.
